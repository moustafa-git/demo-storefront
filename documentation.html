Introduction

SpaceAids is a premium Next.js 15 storefront integrated with Medusa V2, designed specifically for brands offering high-end product customization. The standout feature is a fully interactive 3D product customizer with real-time material and skin-tone matching.

Frontend Stack

  • • Next.js 15 (App Router)
  • • React Three Fiber (Three.js)
  • • GSAP Animations
  • • Tailwind CSS

Backend Stack

  • • Medusa V2 (PostgreSQL)
  • • Stripe Integration
  • • Redis Cache
  • • Supabase Storage (Assets)

Quick Start

To get the project running locally, follow these steps:

  1. Clone and Install: 
    npm install
  2. Environment Setup:

    Copy .env.template to .env and fill in your Medusa backend URL and Stripe keys.

  3. Run Development Server: 
    npm run dev

3D Customizer Logic

"The 3D logic is decoupled from the UI, allowing you to swap GLB models by simply updating the product metadata in the Medusa Admin."

How it works:

The customizer identifies meshes by their name in the GLB file. When a user clicks a part, the system matches the mesh name to the configurable options defined in Medusa.

  • TIP Ensure your GLB meshes are named consistently (e.g., 'primary_shell', 'hinge') to enable targeting.

Partner Program Setup

The Partner Program uses Stripe Pricing Tables and custom Medusa Customer groups.

Implementation Steps:
  • Create your products in Stripe (Tier 1, Tier 2, etc.)
  • Generate a Stripe Pricing Table and get the pricing-table-id
  • Update the ID in src/app/[countryCode]/(main)/distributor-partner/page.tsx
